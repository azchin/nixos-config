* How to modularize
Private stuff in a submodule?
- hardware

  - boot params, amdgpu
- machine specific

  - timezone
  - monitor dpi
  - theming
- programs
  - user stuff
  - configured stuff (e.g. steam)
  - fonts
  - awesomewm override
  - system packages
- extra stuff
  - wireguard
  - fcitx
